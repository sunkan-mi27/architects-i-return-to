Architects I Return To...
A small personal study of four architects whose ideas keep shaping how I think about space — material, light, and care in construction.
Featuring: Peter Zumthor, Zaha Hadid, Demas Nwoko, and Oluwole Olumuyiwa
🔗 Live site — https://sunkan-mi27.github.io/architects-i-return-to/
A single scrolling page — quote, short bio, and key works for each architect. The divider between sections borrows from each architect's own material language: stacked stone courses for Zumthor, a flowing curve for Hadid, a brick lattice for Nwoko, and louvre slats for Olumuyiwa.
Built with

CSS — custom properties, no framework
Vanilla JavaScript — IntersectionObserver for scroll-reveal
